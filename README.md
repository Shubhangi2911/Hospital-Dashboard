
# 🏥 Hospital Analysis Dashboard (Power BI)

## 📌 Objective
The main objective of this project is to design an interactive hospital dashboard using Power BI that provides a centralized view of hospital operations, including patients, doctors, appointments, and financial data.  

The goal is to help management monitor performance, track key metrics, and make data-driven decisions efficiently.

---

## 🧭 Dashboard Overview
This Power BI dashboard is structured into multiple interactive pages:

- Home (Landing Page)
- Overview Dashboard
- Patient Dashboard
- Doctor Dashboard
- Hospital Dashboard
- Finance Dashboard

It transforms raw healthcare data into meaningful insights using visualizations and KPIs.

---

## 🏠 1. Home Page (Landing Page)
- Provides quick introduction and navigation
- Easy access to all modules (Patient, Doctor, Hospital, Finance)
- Includes:
  - Navigation panel
  - Visual hospital theme
  - Page description cards

🎯 **Business Value:**  
Improves user experience and ensures smooth navigation across dashboard sections.

---

## 📊 2. Overview Dashboard
Provides a high-level summary of hospital operations.

### Key Features:
- KPI Summary (Patients, Doctors, Bills, Staff)
- Patient Discharge Trends
- Revenue Breakdown (Charges Type)
- Medicine Tracking
- Patient Feedback & Ratings
- Patient Summary (Admitted vs Discharged)
- Appointment Scheduling

🎯 **Business Value:**  
Helps management quickly understand overall hospital performance.

---

## 🧑‍⚕️ 3. Patient Dashboard
Provides a 360-degree view of individual patient data.

### Key Features:
- Patient Filter (Slicer + Image Selection)
- Patient Profile & Personal Details
- Admission & Discharge Tracking
- Medicine Usage Analysis
- Charges Breakdown
- Total Bill & Medicine Quantity
- Diagnosis & Doctor Information

🎯 **Business Value:**  
Improves patient tracking, treatment analysis, and financial transparency.

---

## 👨‍⚕️ 4. Doctor Dashboard
Focuses on doctor performance and revenue contribution.

### Key Features:
- Doctor Profile (Department, Salary, Specialization)
- Appointment Tracking
- KPI Metrics (Commission, Fees, Patient Spend)
- Commission & Performance Analysis
- Interactive Commission Calculator
- Patient Billing Details
- Doctor Ratings

🎯 **Business Value:**  
Helps evaluate doctor performance and optimize revenue generation.

---

## 🏥 5. Hospital Dashboard
Provides a summarized operational view of the hospital.

### Key Features:
- KPI Summary
- Discharge Trends
- Revenue Analysis
- Medicine Usage Tracking
- Ratings & Feedback
- Patient Summary
- Appointment Insights

🎯 **Business Value:**  
Enables efficient monitoring of hospital operations.

---

## 💰 6. Finance Dashboard
Focuses on financial performance and revenue insights.

### Key Features:
- Financial KPIs (Revenue, Staff Salary, Commission)
- Medicine Stock & Sales Tracking
- Monthly Sales Trends
- Revenue Distribution by Services
- Supplier / Category Analysis
- Payment Mode Filters (Cash, Credit, Insurance)

🎯 **Business Value:**  
Supports financial planning, cost control, and profitability analysis.

---
